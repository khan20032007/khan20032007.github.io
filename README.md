# Bilerplate for gh pages

This is a boilerplate for gh pages. You can use it to create your own website. It includes a profile image, a projects section, and a contact section. You can customize it as you like.

Please note that the projects section is currently fetching data from my github account. You can change it to fetch data from your own github account by changing the `githubUsername` variable in the `projects.js` file.

Also the social media links in the contact section are currently my own. You can add your own social media links there.
